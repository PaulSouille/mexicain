# Mexicain

## Installation
- Clone
- npm install
- Créer un fichier config.js à la racine

```
var config = {};

config.token = {};

config.token.discord = "";
config.token.giphy = "";
config.token.newsapi = "";
config.url = {};
config.url.api = ''
config.url.joiesCode = ''

module.exports = config;
```
